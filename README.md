# 🦸 Super Heroes 30 Days App

The **Super Heroes 30 Days App** is an Android application built using **Kotlin and Jetpack Compose**.
The app displays a list of superheroes along with their **name, description, and image**, showcasing modern Android UI development practices.

This project focuses on implementing **Material Design, Lazy Lists, and UI theming** using Jetpack Compose.

---

## 📱 Features

* 🦸 Displays a **list of superheroes**
* 🖼️ Each hero contains **image, name, and description**
* 📜 Scrollable list using **LazyColumn**
* 🎨 **Material Design 3 UI**
* 🌗 Supports **Light and Dark Theme**
* ⚡ Smooth UI rendering with **Jetpack Compose**

---

## 🛠️ Tech Stack

* **Kotlin**
* **Jetpack Compose**
* **Material Design 3**
* **Android Studio**

---

## 📚 Concepts Used
